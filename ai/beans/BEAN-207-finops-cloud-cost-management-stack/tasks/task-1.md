## Description

Create the finops cloud cost management stack in `ai-team-library/stacks/finops/` as a multi-file directory covering:

1. **cost-optimization.md** — Cloud spend optimization, waste identification, right-sizing recommendations
2. **tagging-allocation.md** — Tagging strategies, cost allocation models, showback/chargeback reporting
3. **budgets-planning.md** — Budget alerts, reserved instance planning, commitment-based discounts

Each file must follow the standardized template with: Defaults table with alternatives, Do/Don't lists, Common Pitfalls, Checklist, and code examples.
